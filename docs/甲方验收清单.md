# 甲方验收清单

## 1. 用户体系

- 验收操作：打开用户中心，创建或切换 dev_mock 用户。
- 预期结果：接口自动携带 `X-User-Id`，不同用户只看到自己的项目和角色。
- 当前限制：不是正式微信登录。

## 2. 剧本项目空间

- 验收操作：创建两个剧本项目并分别上传资料。
- 预期结果：项目 summary 显示文件、解析文档、角色和知识片段数量。
- 当前限制：暂不支持正式团队协作和权限分级。

## 3. 文件上传与解析

- 验收操作：上传 TXT / DOCX / PDF 并解析。
- 预期结果：生成 parsed document，显示摘要、字数和解析状态。
- 当前限制：复杂版式 PDF 仍可能需要后续增强。

## 4. OCR

- 验收操作：上传 PNG / JPG 图片。
- 预期结果：默认走 mock OCR，ParsedDocument 可进入角色生成和知识库链路。
- 当前限制：真实 OCR 需要部署环境安装并配置 Provider。

## 5. 角色候选识别

- 验收操作：在项目详情页点击识别角色候选。
- 预期结果：识别出规则命中的候选角色，可编辑或忽略。
- 当前限制：当前不是 LLM 抽取，识别质量依赖文本格式和规则。

## 6. 角色关系图

- 验收操作：点击抽取角色关系。
- 预期结果：生成关系列表和简易关系图，可编辑或忽略。
- 当前限制：当前为规则抽取，不是复杂图数据库。

## 7. 角色生成

- 验收操作：通过单角色或批量方式生成数字人。
- 预期结果：生成 Character、CharacterProfile、KnowledgeChunk、Pet、PetAsset、默认调教配置。
- 当前限制：角色设定为规则抽取 + mock 补全。

## 8. 知识库检索

- 验收操作：在同名角色的不同项目中分别聊天。
- 预期结果：只检索当前 project_id 下的知识片段。
- 当前限制：当前为关键词检索，未接向量数据库。

## 9. 聊天

- 验收操作：进入聊天页发送普通消息和关系问题。
- 预期结果：返回角色化回复、pet_action、provider_debug，并保存会话消息。
- 当前限制：未配置 API Key 时使用 MockLLMProvider。

## 10. 长期记忆

- 验收操作：发送“我喜欢雨夜散步”，再查看记忆页。
- 预期结果：自动保存 user_preference 记忆，后续聊天注入 Prompt。
- 当前限制：当前为规则抽取，不是 LLM 记忆总结。

## 11. 角色调教

- 验收操作：修改语气、称呼、回复长度、剧透保护、桌宠开关。
- 预期结果：保存配置后聊天 Prompt 和 mock 回复轻量响应配置。
- 当前限制：这是 Prompt-level customization，不是真正模型微调。

## 12. Q版桌宠

- 验收操作：进入桌宠资产页生成并切换 active asset。
- 预期结果：聊天页优先使用 active_pet_asset 图片，pet_enabled=false 时隐藏桌宠。
- 当前限制：当前默认 mock 图片，不是真实 AI 绘图或 Live2D。

## 13. 内容安全

- 验收操作：发送自伤、Prompt 注入、剧透追问等测试输入。
- 预期结果：触发安全回复或 warn，不进入普通恋陪暧昧回复。
- 当前限制：当前为规则版 safety，不是真实审核 API。

## 14. 微信小程序骨架

- 验收操作：运行 `npm run build:mp-weixin`。
- 预期结果：构建成功，可导入微信开发者工具。
- 当前限制：未接正式微信登录、合法域名和审核流程。

## 15. 自动化测试

- 验收操作：运行 `tests/run_backend_smoke.py`、`tests/run_project_isolation.py`、`tests/run_safety_checks.py`。
- 预期结果：脚本 PASS 并生成 JSON 报告。
- 当前限制：当前为项目级 smoke test，不是完整 CI。

## 16. 存储与部署骨架

- 验收操作：查看 `/api/storage/status` 和 `/api/system/health`。
- 预期结果：显示 local / Provider 状态，不暴露密钥。
- 当前限制：生产对象存储和云数据库需要部署时配置。
